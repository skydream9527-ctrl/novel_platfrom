网页研究能力：从预设信息源或指定 URL 抓取内容并提炼知识。

工作流程：
1. 接收 URL 或使用预设信息源列表
2. 抓取网页正文
3. 提炼事实、建议、风险和讨论点
4. 与已有知识卡片比对
5. 输出四层结构：事实层、建议层、变化层、讨论层
