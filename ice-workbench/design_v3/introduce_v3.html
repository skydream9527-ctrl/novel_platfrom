CORE CONCEPTS

核心概念

理解 ICE Data Workbench 的六大核心

📋

任务 (Task)

用户发起的一次数据工作流，每个任务有独立工作空间，记录完整对话历史和文件产出。

🤖

Agent

公共智能体，与工作范式 1:1 绑定。通过对话持续进化，经验卡片注入使其越用越聪明。

⚡

Skill

可执行工具，具备 tool_schema。Agent 通过 function calling 调用 Skill 完成数据查询、分析、可视化。

✨

经验卡片

从对话中自动提炼的规则与知识。经审批后注入 Agent 上下文，驱动 Agent 能力进化。

🎯

工作范式

5 种预置任务模板 — AB 实验、经营分析、版本灰度、数据分析、波动分析，绑定专属 Agent + Skills。

🌐

公共区

团队共享的 Agents、Skills、Files 和任务模板。知识沉淀与复用的核心。
