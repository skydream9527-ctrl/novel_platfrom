# Topic Card: Agent Frameworks

## Scope

研究 AI Agent 框架、工作流组织方式、工具调用设计和多 Agent 协作边界。

## Current Understanding

- 框架选型不能只看功能数量，要看可维护性、可观测性和调试体验
- workflow 与 autonomous agent 不是替代关系，而是适用边界不同
- 很多场景下，简单 workflow 比复杂多 Agent 更稳定

## Evidence

- 来自已学习的飞书文档 1 和 2

## Open Questions

- 什么时候应该从 workflow 升级到 agent
- 多 Agent 的收益是否覆盖复杂度成本
- 选框架时最关键的评估维度应该如何排序
