# Topic Card: Tool Use Patterns

## Scope

研究模型调用工具、外部系统接入、MCP、任务编排和可靠性设计模式。

## Current Understanding

- 工具调用设计的重点不是接入数量，而是成功率、可恢复性和边界控制
- 可靠的工具调用通常需要明确输入约束、错误处理和结果校验
- 学习型 Agent 需要把工具调用本身也作为知识对象来维护

## Open Questions

- MCP 适合哪些场景，不适合哪些场景
- 工具失败时的降级策略应该如何统一
