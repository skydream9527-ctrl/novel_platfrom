## 与决策的关联

- **D53**：公共文件**防误删** — Sidebar 公共文件分组下不显示 `[×]` 删除按钮
- **D93**：编辑权限统一到 admin（`/admin/files`），普通用户只读
- **D120**：admin 后台支持上传 + 文本类（md/txt）内容编辑 + `is_pinned` 置顶
- **D121**：与任务模板的"file_seeds" 字段联动（模板可预填这里的公共文件作为初始上下文）

## 元数据

公共文件的元数据（`is_pinned / sync_source / updated_at` 等）存储在：

- 文件优先架构（D134）下：`.meta.json`（隐藏文件）与每份 md 同目录
- 或集中：`files/.metadata/index.json`

具体方式由后端实现确定（见 BACKEND.md）。
