# ICE Data Workbench 使用指南

> 欢迎使用 ICE Data Workbench — AI 驱动的数据工作流工作台
> 本指南帮助你快速上手，了解全部核心功能和操作方法

---

## 一、快速开始

### 1.1 登录系统

1. 打开系统地址，进入登录页
2. 选择你的账号点击登录，或输入邮箱和密码
3. 支持飞书账号一键登录（需管理员开通）

### 1.2 创建第一个任务

1. 登录后进入 Dashboard 首页
2. 在"创建新任务"区域选择一个工作范式：
   - ⚖ **AB 实验** — 实验设计、分流配置、指标分析、结论产出
   - 📈 **经营分析** — 经营指标拆解、归因、趋势洞察
   - 🔌 **版本灰度** — 灰度放量监控、质量对比、放量决策
   - 📊 **数据分析** — 通用数据探查、SQL 查询、建模、可视化
   - 🔥 **波动分析** — 核心指标异动检测、归因定位、影响评估
3. 输入任务名称，点击"创建任务"
4. 系统自动为你配置对应的 Agent 和 Skills，直接进入工作空间

> 💡 如果你的需求不属于以上范式，可以选择"创建开放任务"，自由选择 Agent

## 二、工作空间

### 2.1 界面布局

工作空间采用三栏布局：

```
┌─────────────────────────────────────────────────────────┐
│ 顶部导航：任务名称 · 范式标签 · 操作按钮                    │
├──────────┬─────────────────────────────┬────────────────┤
│ 左侧边栏 │       聊天对话区             │  右侧预览面板   │
│          │                             │               │
│ Agent    │  你的消息 / Agent 回复       │  文件预览      │
│ Skills   │  工具执行状态               │  知识库文档    │
│ Files    │  图表 · 报告 · 追问建议      │               │
│ 知识库   │                             │               │
└──────────┴─────────────────────────────┴────────────────┘
```

### 2.2 与 Agent 对话

1. 在底部输入框输入你的需求，按 Enter 发送
2. Agent 会理解你的意图，调用 Skills 执行数据查询和分析
3. 结果以文字、表格、图表等形式呈现在对话中
4. 你可以继续追问或点击"推荐追问"按钮深入分析

**快捷操作：**
- `Enter` — 发送消息
- `Shift + Enter` — 换行
- `⌘K` — 全局搜索

### 2.3 引用文件和技能

在输入框中可以引用资源来增强 Agent 的能力：

- 📎 **引用文件** — 点击后选择工作空间中的文件，内容会注入 Agent 上下文
- ⚡ **引用技能** — 点击后选择特定 Skill，Agent 将通过 Tool Calling 调用该工具
- 📤 **上传文件** — 从本地上传 csv/md/txt 等文件作为分析输入

### 2.4 文件管理

**查看文件：** 点击左侧边栏的文件名，右侧面板展示预览

**上传文件：** 点击 Files 区域的 "+" 按钮，或直接拖拽文件到聊天区

**下载文件：** 在文件预览面板点击 ⬇ 下载按钮

**文件分类：**
- **工作产出** — Agent 生成的分析报告、图表、数据表
- **需求输入** — 你上传的需求文档、原始数据、背景资料

### 2.5 知识库

侧边栏底部展示已连接的知识库目录：
- 📖 **飞书知识库** — 自动每日同步飞书知识空间的文档
- 🧠 **RAG 知识库** — 连接 Mify RAG 语义检索

点击知识库名称展开完整目录树，点击文档可预览内容。

点击 🔄 按钮可手动刷新同步最新内容。

## 三、Agent 与 Skills

### 3.1 什么是 Agent

Agent 是你的智能助手，每种工作范式绑定一个专属 Agent：
- Agent 具备领域知识和分析能力
- Agent 通过 Tool Calling 调用 Skills 执行具体操作
- Agent 在每次对话中学习进化，积累经验

### 3.2 什么是 Skill

Skill 是 Agent 可调用的工具能力，例如：
- **SQL 查询** — 连接数据仓库执行查询
- **指标计算** — 统计指标的计算和对比
- **趋势分析** — 时间序列趋势识别
- **归因分析** — 多维度变化归因
- **报告生成** — 结构化分析报告输出
- **可视化** — 图表生成

### 3.3 Agent 进化

Agent 会从你的对话中学习：
1. 高质量的分析路径和结论会被记录为 Agent 记忆
2. 经过验证的记忆会升级为"经验卡片"
3. 管理员审批后，经验卡片注入 Agent 的知识库
4. Agent 在后续对话中自动应用这些经验

你可以通过以下方式加速 Agent 进化：
- 对好的回答点 👍
- 点击"💡 沉淀为经验"手动标记有价值的内容

## 四、公共区

公共区是团队共享的资源池，包含：

| 类型 | 说明 |
|------|------|
| **Agents** | 团队沉淀的可复用 Agent |
| **Skills** | 可复用的技能模块 |
| **Files** | 经验文档、规范文档、模板 |
| **模板** | 预配置的任务模板，一键使用 |

在公共区你可以：
- 浏览和搜索团队资源
- 查看 Agent/Skill 的详细信息
- 使用任务模板快速创建同类任务

## 五、进阶功能

### 5.1 模型切换

在聊天区顶部的模型选择器中，你可以切换不同的 LLM 模型：
- **GPT-5.4** — 推荐，日常分析
- **Claude Opus 4.6** — 强力，复杂推理
- **GPT-5.5** — 最新能力
- 更多模型可选...

### 5.2 导出报告

点击顶栏"📤 导出"按钮：
1. 选择导出范围（整个对话 / 选择消息）
2. 选择报告格式（结构化报告 / 原始对话）
3. 系统会将对话整理为结构化分析报告并保存为文件

### 5.3 任务协作

点击顶栏"🔗 分享"按钮：
1. 搜索并添加协作者
2. 设置权限：只读（查看）或协作（参与对话）
3. 被分享者在 Dashboard 会看到带"协作"标签的任务

### 5.4 定时任务

创建任务时可设为定时任务：
- 配置执行频率（每日/每周/自定义）
- 设置执行时间和 Prompt
- 选择结果推送方式（保存文件/飞书通知/邮件）
- Agent 会按计划自动执行分析并推送结果

### 5.5 全局搜索

按 `⌘K`（Mac）或 `Ctrl+K` 打开全局搜索：
- 搜索历史任务
- 搜索 Agents 和 Skills
- 搜索文件内容

## 六、常见问题

### Q: Agent 回复不准确怎么办？

1. 尝试更清晰地描述需求，提供具体的指标名称和时间范围
2. 引用相关文件（如指标口径文档）让 Agent 理解上下文
3. 切换到更强的模型（如 Claude Opus 或 GPT-5.4 Pro）
4. 对不好的回答点 👎，帮助 Agent 学习

### Q: 如何上传数据进行分析？

1. 点击输入框的 📤 按钮或侧边栏 Files 的 "+" 按钮
2. 支持格式：csv, md, txt, html, json, sql
3. 文件大小限制：2MB
4. CSV 文件会自动解析列结构，供 Agent 分析引用

### Q: 如何查看飞书知识库的最新内容？

1. 系统每天自动同步一次飞书知识库
2. 点击知识库旁的 🔄 按钮可手动刷新
3. 展开目录树查看完整文档列表
4. 点击文档名称可预览内容

### Q: 我的对话数据安全吗？

- 所有对话数据加密存储在系统数据库中
- 对话仅任务创建者和被授权的协作者可见
- 管理员无法查看用户对话内容
- 定期备份确保数据不丢失
