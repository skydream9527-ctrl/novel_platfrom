## 使用说明
1. 搜索核心指标通常包括：
   - 搜索DAU（日活跃搜索用户数）
   - 搜索次数（Search Count）
   - 搜索点击率（CTR）
   - 搜索结果点击分布
   - 搜索转化率
   - 搜索建议使用率
   - 搜索失败率

2. 常用的数据源可能包括：
   - 搜索事件表
   - 搜索结果点击表
   - 搜索建议曝光表
   - 搜索转化表

## 注意事项
- 搜索指标需要区分不同搜索场景（如网页搜索、图片搜索、视频搜索等）
- 需要考虑搜索质量指标（如搜索满意度、搜索成功率等）
