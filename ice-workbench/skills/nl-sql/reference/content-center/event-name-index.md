## Usage

When user provides an event name for埋点数据 query:

1. **Search** the event name in this index
2. **If found**: Continue to generate SQL
3. **If not found**: Inform user the event is not supported

### Example Validation

| User Input | Status | Action |
|------------|--------|--------|
| `content_item_click` | ✅ Found (#13) | Generate SQL |
| `content_item_expose` | ✅ Found (#1) | Generate SQL |
| `app_open` | ✅ Found (#182) | Generate SQL |
| `unknown_event` | ❌ Not found | Stop and inform user |
